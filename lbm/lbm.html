Lattice Boltzmann Method View on GitHub
The library offers a GPU accelerated lattice Boltzmann method (LBM) implementation. It uses the BGK operator to calculate the equilibrium and the D2Q9 velocity set. Examples of several boundary conditions are provided:
  • Bounce back
  • Bounce forward
  • Open inlet/outlet
Fluid-structure interaction can be simulated via the immersed boundary method. Below are a couple of study cases that illustrate the integration of these capabilities.

  • Flow around a cylinder

    [source code]
    This study case uses the multi direct-forcing IB-LBM to simulate the interaction between the fluid and a cylinder fixed in place. The plots are in lattice units.

  • Poiseuille flow

    [source code]
    This example is a pressure driven flow with bounce back conditions at the top and bottom boundaries to model no-slip walls. Another example uses gravity as the driving mechanism. The plots are in lattice units.
